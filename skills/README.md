# Skills — 절차적 메모리

경험에서 추출한 재사용 가능한 절차들. 에이전트가 작업을 마칠 때마다 새 패턴을 여기에 증류한다.

## 구조

```
skills/
├── file-ops/      # 파일 작업, 변환, 정리
├── research/      # 웹 리서치, 정보 수집, 요약
├── writing/       # 문서 작성, 번역, 편집
└── <새 카테고리>/ # 필요에 따라 추가
```

## 스킬 인덱스

| 파일 | 설명 | 마지막 개선 |
|------|------|------------|
| *(아직 없음 — 작업 후 자동 추가됨)* | | |

## 스킬 추가 기준

다음 조건이 하나라도 맞으면 스킬 파일 생성:
- 같은 유형의 작업을 2번 이상 함
- 단계가 3개 이상이고 순서가 중요한 작업
- 다음에 더 빠르게 할 수 있는 비결이 있는 작업
- 함정(gotcha)이 있어서 기억해야 하는 작업
