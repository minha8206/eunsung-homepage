# Sessions — 세션 로그

세션별 주요 결정, 완료된 작업, 발견한 것들을 기록한다.
FTS(전문 검색)처럼 쓴다 — "저번에 ~했던 거 어떻게 했지?" 에 답하기 위해.

## 로그 형식

파일명: `YYYY-MM-DD_간략설명.md`

```markdown
# 2026-06-16 — 세션 제목

**완료한 작업**: 
**새로 배운 것**: 
**생성된 스킬**: 
**업데이트된 메모리**: 
**다음 세션 이어서 할 것**: 
```
