# Projects — 프로젝트 컨텍스트

진행 중인 프로젝트별 맥락, 목표, 현재 상태를 기록한다.
새 작업이 들어오면 관련 프로젝트 파일을 먼저 읽는다.

## 프로젝트 파일 형식

파일명: `<프로젝트명>.md`

```markdown
# 프로젝트명

**목표**: 
**현재 상태**: 
**다음 할 일**: 
**관련 스킬**: skills/카테고리/파일명.md
**마지막 업데이트**: YYYY-MM-DD

## 맥락

프로젝트 배경, 제약사항, 중요한 결정들

## 완료된 마일스톤

- [ ] 마일스톤 1
```
