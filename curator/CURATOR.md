# Curator — 스킬 & 메모리 품질 관리

Hermes의 Curator 패턴을 구현한다. 주기적으로 스킬 라이브러리와 메모리를 점수화, 병합, 가지치기한다.

## Curator 루틴

**주기**: 주 1회 (또는 스킬이 10개 이상 쌓였을 때)

**작업 목록**:

1. **점수화** — 각 스킬의 사용 횟수와 최근성 확인
   - 3개월 이상 사용 안 됨 → 아카이브 후보
   - 오류 보고 있음 → 수정 우선순위

2. **병합** — 유사한 스킬 통합
   - 80% 이상 겹치는 스킬 → 하나로 합치기
   - 파생 스킬이 생겼으면 → 상위 스킬 업데이트

3. **가지치기** — 불필요한 항목 제거
   - 중복 메모리 항목 → 하나만 남기기
   - 틀린 정보 → 즉시 삭제

4. **개선** — 자주 쓰이는 스킬 품질 향상
   - 사용 횟수 상위 3개 스킬 → 절차 검토 및 개선
   - 주의사항 섹션 보강
